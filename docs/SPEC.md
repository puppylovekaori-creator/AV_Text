# AV_Text 仕様メモ

## 対象
- Firefox アドオン `AV Text Paster`
- Native Messaging host `avtext_helper.py`
- 保存先 `actress.txt` / `title.txt` / `no.txt`

## 右クリックメニュー仕様
- `タイトルに送る`
  - 選択文字列を単一行に正規化して `title.txt` を上書きする
- `女優名に送る`
  - 選択文字列を単一行に正規化して `actress.txt` を全置換する
- `女優名に送る（別名付き）`
  - 選択文字列を単一行に正規化し、末尾へ `()` を付けて `actress.txt` を全置換する
- `女優に追加`
  - 選択文字列の前後空白と改行を除去し、`actress.txt` の末尾へ追記する
  - 既存内容が空でなければ、既存末尾と追加文字列の間に半角スペースを 1 個だけ入れる
- `女優に別名で追加`
  - `女優に追加` と同じ追記動作で、追加する文字列の末尾に `()` を付ける
- `品番に送る`
  - 選択文字列を単一行に正規化して `no.txt` を全置換する
- `サクラエディタを前面に`
  - AV Text 専用入力エディタを前面に出す

## 追記時の補足
- `actress.txt` が空なら先頭スペースは入れない
- `actress.txt` の既存内容も単一行へ正規化してから追記結果を再保存する
- 保存時は UTF-8 / LF を使う
